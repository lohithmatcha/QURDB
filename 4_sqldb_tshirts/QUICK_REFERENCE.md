# 🚀 QurdDB - Quick Reference Card

## ⚡ ELEVATOR PITCH (30 seconds)
"QurdDB is an intelligent database assistant that lets anyone ask questions in natural language - no SQL knowledge required. You can type or speak your questions, and get instant insights with beautiful visualizations."

---

## 🎯 KEY TECHNOLOGIES USED

### **AI/ML**
- Google Gemini 2.5 Flash (LLM)
- LangChain (Orchestration)
- HuggingFace Embeddings
- ChromaDB (Vector Store)
- Few-Shot Learning

### **Database**
- MySQL
- PyMySQL connector

### **Frontend**
- Streamlit (Web UI)
- Plotly (Visualizations)
- Web Speech API (Voice)
- HTML/CSS/JavaScript

### **Data Processing**
- Pandas
- Plotly Express

## ⚠️ TROUBLESHOOTING

**If voice doesn't work:**
- Use Chrome, Edge, or Safari
- Allow microphone permissions
- Have backup text questions ready

**If query fails:**
- Show example questions
- Explain it's a learning system
- Offer to refine the question
